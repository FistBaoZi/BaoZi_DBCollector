数据采集系统操作手册

1. 系统登录

1.1 访问系统登录页面

1.2 输入用户名和密码

1.3 输入验证码（点击验证码图片可刷新）

1.4 点击登录按钮进入系统

注意：若验证码无法识别，可点击验证码图片进行刷新

2. 厂站管理

2.1 添加厂站

  • 点击右上角"添加厂站"按钮
  • 填写厂站名称
  • 填写厂站ID（唯一标识）
  • 填写数据库连接字符串
  • 点击确定保存

2.2 编辑厂站

  • 在厂站列表中找到需要编辑的厂站
  • 点击"编辑"按钮
  • 修改相关信息
  • 点击确定保存

2.3 删除厂站

  • 在厂站列表中找到需要删除的厂站
  • 点击"删除"按钮
  • 在确认对话框中点击确定

3. 采集配置

3.1 配置采集表

  • 在厂站列表中点击"采集配置"按钮
  • 在左侧列表中选择需要采集的数据表
  • 点击中间的向右箭头将表添加到右侧
  • 可以使用搜索功能快速查找表
  • 点击右上角"保存配置"按钮保存设置
提示：
  • 左侧显示所有可选的数据表
  • 右侧显示已配置要采集的数据表
  • 可以随时调整配置，添加或移除需要采集的表

4. 连接字符串格式说明

标准格式：

Data Source=服务器地址;Initial Catalog=数据库名;User ID=用户名;Password=密码;

示例：

Data Source=.;Initial Catalog=DBName;User ID=sa;Password=123456;
注意：请确保提供的数据库连接信息正确，且数据库用户具有适当的访问权限。

5. 常见问题

5.1 无法连接数据库

  • 检查连接字符串格式是否正确
  • 确认数据库服务器是否可访问
  • 验证用户名和密码是否正确
  • 检查网络连接是否正常

5.2 采集配置不生效

  • 确保已点击"保存配置"按钮
  • 检查数据库用户是否有相应表的读取权限
  • 刷新页面查看最新配置
